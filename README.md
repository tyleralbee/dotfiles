# dotfiles

This repository contains my personal dotfiles (configurations) for the tools and applications I use.

## Tools

### Meta

Tools I use to manage this repository and my dotfiles:

- [stow](https://www.gnu.org/software/stow/manual/stow.html) (symlink-based dotfile management)
- [pnpm](https://pnpm.io/) (package management for the devtools I use to manage this repository)
- [prettier](https://prettier.io/) (formatting)

### CLI

Command-line tools:

- [bash](https://www.gnu.org/software/bash/manual/bash.html): my default shell
- [cheat](https://github.com/cheat/cheat): cheatsheets
- [complete](https://www.gnu.org/software/bash/manual/bash.html#Programmable-Completion): bash tab completion
- [fontconfig](https://fontconfig.pages.freedesktop.org/fontconfig/fontconfig-user.html): font configuration
- [gh](https://cli.github.com/manual/gh_config): GitHub CLI
- [git](https://git-scm.com/doc): version control
- [kanata](https://github.com/jtroo/kanata/blob/main/docs/config.adoc): keyboard remapping
- [pg_format](https://github.com/darold/pgFormatter/blob/master/doc/pg_format.pod): PostgreSQL SQL syntax beautifier
- [readline](https://www.gnu.org/software/bash/manual/html_node/Readline-Init-File.html): bash input
- [ripgrep](https://github.com/BurntSushi/ripgrep/blob/master/GUIDE.md): file search

### TUI

Terminal user interfaces (TUIs):

- [nvim](https://neovim.io/doc/user/): text editor
- [tmux](https://github.com/tmux/tmux/wiki): terminal multiplexer
- [yazi](https://yazi-rs.github.io/docs/configuration/overview): terminal file manager

### GUI

- [fuzzel](https://codeberg.org/dnkl/fuzzel/src/branch/master/doc/fuzzel.ini.5.scd): app launcher
- [ghostty](https://ghostty.org/docs/config): terminal emulator
- [hypr](https://wiki.hypr.land/Configuring/): Wayland compositor
- [obsidian](https://help.obsidian.md/): notes
- [tridactyl](https://tridactyl.xyz/build/static/docs/modules/_src_excmds_.html): Firefox Vim motions
- [waybar](https://github.com/Alexays/Waybar/wiki/Configuration): status bar
- [zed](https://zed.dev/docs/): IDE
